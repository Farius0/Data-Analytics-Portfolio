# FIFA Player Performance Analytics (2015–2022)

Structured analytical project exploring player performance, market valuation, and statistical evolution across FIFA datasets (2015–2022).

This project demonstrates the ability to transform large sports datasets into structured analytical insights using Python-based workflows.

---

## Project Objective

To analyze how player attributes, market value, and performance indicators evolve over time and interact with each other.

The project focuses on:

- Metric engineering
- Statistical correlation analysis
- Time-series evolution
- Value-performance alignment
- Structured analytical interpretation

---

## Data Sources

- FIFA 15 → FIFA 22 player datasets
- Historical World Cup data (1930–2022)

Data preparation included:

- Variable standardization
- Feature selection
- Missing value handling
- Dataset consolidation
- Structured preprocessing pipeline

---

## Analytical Scope

### 1️⃣ Player Performance Evolution

- Global trend analysis
- Attribute progression over seasons
- Focused analysis on selected players

### 2️⃣ Attribute Correlation Analysis

- Correlation between player attributes (speed, shooting, defense, etc.)
- Relationship between attributes and overall rating
- Identification of key drivers of performance score

### 3️⃣ Market Value Dynamics

- Transfer movement analysis
- Evolution of player market value
- Value volatility across seasons

### 4️⃣ Value vs Performance Alignment

- Relationship between market value and team performance
- Relationship between player rating and real-world outcomes
- Identification of valuation inefficiencies

---

## Analytical Techniques Used

- Pandas data transformation
- Correlation matrices
- Time-series comparison
- GroupBy aggregations
- Feature engineering
- Statistical interpretation

---

## Key Insights

- Strong correlation clusters between technical attributes
- Market value influenced by both performance and external factors
- Variability in valuation efficiency across seasons
- Evolution of player profiles over time
